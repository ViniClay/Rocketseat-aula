Operadores aritméticos:

  • == igual
  • != diferente
  • === estritamente igual
  • !== estritamente diferente
  • > maior
  • >= maior ou igual
  • < menor
  • <= menor ou igual

Operadores de atribuição:

  • += recebe soma (x += 2 => x = x + 2)
  • -= recebe subtração (x -= 2 => x = x - 2)
  • *= recebe multiplicação (x *= 2 => x = x * 2)
  • /= recebe divisão (x /= 2 => x = x / 2)
  • **= recebe potência (x **= 2 => x = x ** 2)

Operadore lógicos:

  • && (AND)
  • || (OR)
  • ! (NOT)
  • Condição: condition ? ifTrue : ifFalse

Falsy:

  • false
  • 0
  • -0
  • ""
  • null
  • undefined
  • NaN

Truthy:

  • true
  • {}
  • []
  • number
  • string (não nula)
  • Infinity
  • -Infinity

Precedência de operadores:

  • Grouping () ;
  • Negação e incremento !, ++, -- ;
  • Multiplicação e divisão *, / ;
  • Adição e subtração +, - ;
  • Relacional <, <=, >, >=, c ;
  • Igualdade ==, !=, ===, !== ;
  • And && ;
  • Or || ;
  • Condicional ?: ;
  • Assigment (atribuição) =, +=, -=, *= ;
